# 📊 SQL Employee Queries Practice

This repository contains essential SQL query concepts performed on an employee dataset.

---

##  Dataset

The dataset consists of employee details with the following attributes:

* Employee ID
* Name
* Department
* Salary
* Age

---

##  Arithmetic Operations

Perform calculations on columns such as:

* Annual salary (monthly salary × 12)
* Remainder of age when divided by 2

---

##  Aggregate Functions

Apply aggregate functions to analyze overall data:

* Count total number of employees
* Calculate total salary payout
* Find the maximum age

---

##  Combined Calculations

Compute overall business metrics:

* Total annual salary cost of the company

---

##  Grouping and Filtering

Analyze department-wise data:

* Count employees in each department
* Calculate average salary per department
* Filter departments based on salary conditions

---

##  Purpose

This project helps in understanding:

* SQL arithmetic operations
* Aggregate functions
* Grouping and filtering techniques

---

##  Learning Outcome

By completing this practice, you will be able to:

* Perform calculations using SQL
* Analyze data using aggregate functions
* Work with GROUP BY and HAVING clauses
